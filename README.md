# gin-init
gin脚手架，供学习使用

# 目录介绍
```
 api // 接口
 config // 配置
 core // 核心
 global // 全局
 initialize // 初始化相关
 logs // 日志
 middleware // 中间件
 model // 实体和表结构一一对应   
    dto // 数据传输对象
    request // 请求参数结构体
    response // 返回参数结构、返回函数封装
 router // 路由
 test // 单元测试目录
 utils // 工具包
 config.yaml // 配置信息
```
